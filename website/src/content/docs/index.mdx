Pompelmi is a **TypeScript library** that adds a layered security layer to file upload endpoints. It scans file bytes **in-process** — no HTTP call to a cloud API, no ClamAV daemon, no file data leaving your infrastructure.

Give it a `Buffer`, get back a `clean | suspicious | malicious` verdict in milliseconds. Your middleware, your storage, your rules.

**Made for teams that cannot or choose not to route user files through a third-party service** — healthcare, legal, finance, self-hosted SaaS, and privacy-first products.

> **Live demo:** drag a clean JPG → **CLEAN**. Try the EICAR test string → **MALICIOUS**.

## What Pompelmi checks

Every file passes through a configurable chain of guards before your handler ever touches it:

| Layer | What it catches |
|---|---|
| **Size limit** | Files over your configured threshold |
| **Extension allowlist** | Unexpected file types for your context |
| **Magic-byte MIME validation** | Extensions that lie about their content |
| **Structural heuristics** | PE/ELF headers, embedded scripts, polyglot files |
| **ZIP bomb protection** | Entry count, per-entry size, nesting depth |
| **YARA rules** *(optional)* | Custom or [premium](./enterprise/) signature matching |

All checks return a single `ScanReport` — `verdict`, `reasons`, `durationMs`, SHA-256 — that you act on immediately.

## Is Pompelmi right for your project?

**Yes — if:**
- Your app accepts user-uploaded files and you need to block malicious ones.
- You are in a **regulated or self-hosted environment** where data residency matters (HIPAA, SOC 2, ISO 27001, GDPR).
- You want a composable, testable security layer integrated into your existing Node.js middleware.
- Your team prefers owning the dependency over calling a cloud API.

**Consider alternatives if:**
- You need full AV coverage with live signature updates → pair Pompelmi with `@pompelmi/engine-clamav`.
- You already use a trusted cloud scanning service and data residency is not a concern.

## Policy packs — sensible defaults in one line

```js
import { scanBytes, STRICT_PUBLIC_UPLOAD, IMAGES_ONLY, DOCUMENTS_ONLY } from 'pompelmi';

// Anonymous uploads from the public internet
const result = await scanBytes(buffer, { policy: STRICT_PUBLIC_UPLOAD });

// Avatar endpoint
const result = await scanBytes(buffer, { policy: IMAGES_ONLY });

// Document portal
const result = await scanBytes(buffer, { policy: DOCUMENTS_ONLY });
```

| Policy | Best for |
|---|---|
| `STRICT_PUBLIC_UPLOAD` | Anonymous or untrusted uploaders |
| `CONSERVATIVE_DEFAULT` | General-purpose hardened default |
| `DOCUMENTS_ONLY` | PDF, Word, Excel, CSV portals |
| `IMAGES_ONLY` | Avatar or image-only endpoints |
| `ARCHIVES` | Archive handling with ZIP bomb protection |

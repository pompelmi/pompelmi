Pompelmi adds **safe, blazing‑fast scanning** to your file uploads.

- **UI** — Ready‑made upload button & dropzone for **React/Next.js**.
- **Server** — Lightweight middlewares for **Express/Koa** and helpers for **Next.js API routes**.
- **Safety** — Size & MIME guards, optional **YARA** rules, and clear **CLEAN / SUSPICIOUS / MALICIOUS** verdicts.

> 🧪 **Live demo:** on the homepage drag a clean JPG → **CLEAN**; try the **EICAR** test string → **MALICIOUS**.

## Highlights

- ✅ **Developer‑friendly**: copy‑paste examples, sensible defaults.
- ⚡ **Fast path**: avoids heavy engines unless needed; instant verdicts for common cases.
- 🛡️ **Defense‑in‑depth**: layered checks (extension, MIME, size, optional YARA).
- 🧩 **Modular**: UI and server parts work together or stand‑alone.
- 🔍 **Transparent results**: simple JSON verdict with reasons.

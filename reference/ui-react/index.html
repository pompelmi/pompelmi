@pompelmi/ui-react API

This page documents the React UI components shipped by @pompelmi/ui-react.

Minimum React version: 18 (works with React 19 as well). The package exposes both ESM and CJS builds.

<UploadButton />

Section titled “<UploadButton />”

Renders a clickable control that opens the native file picker and uploads the selected file to your scan engine.

Additional behavior: none (uses only common props).

Accessibility

Section titled “Accessibility”
  • Provide a clear label where appropriate.
  • Communicate verdicts (CLEAN / MALICIOUS) in your UI using visible text or ARIA live regions if needed.

Troubleshooting

Section titled “Troubleshooting”
  • Network/CORS errors → Confirm action points to your engine and that CORS allows your site origin.
  • Large files rejected → Increase maxSize (client) and verify server limits.
  • No callbacks firing → Verify you’re importing from @pompelmi/ui-react and passing a non-empty action.
